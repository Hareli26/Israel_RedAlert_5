# 🔴 Red Alert Monitor v5.0 — התרעות צבע אדום

> **נבנה על ידי הראלי דודאי — מרץ 2026**

מערכת ניטור התרעות פיקוד העורף בזמן אמת לשולחן העבודה של Windows.
שואב מידע מ-API הרשמי של פיקוד העורף (OREF) ומציג התרעות בממשק גרפי מלא.

## ✨ תכונות עיקריות

### 🚨 התרעות בזמן אמת
- שאילתה לשרת פיקוד העורף כל **2 שניות** (ניתן לכוון: 1–10 שניות)
- **סינון לפי ישוב** — קבלת התרעות רק לאזור שלך
- סינון אוטומטי של התרעות בדיקה (`cat=101`)
- תמיכה בסוגי אירועים: טיל / כלי טיס / חדירת מחבלים / חומרים מסוכנים / רעידת אדמה / צונאמי

### 💥 תוצאות נפילה (v4.2)

לאחר התרעת **ירי רקטות וטילים** או **חדירת כלי טייס**:

```
קבלת התרעה (cat 1 / 2 / 13)
     ↓
thread רקע — ממתין 2 דקות
     ↓
שאילתה ל-RSS: ynet + walla (ביטחון)
     ↓
סינון לפי: שמות הישובים בהתרעה + מילות מפתח
(נפילה / מכה / פגיעה / ירוט / כיפת ברזל / תקיפה...)
     ↓
הודעת Tray + FallResultsWindow
```

המידע מוצג עם כותרת הידיעה, תקציר, ושעת פרסום.

### 📱 Telegram Bot (v5.0)

**הגדרה:** הגדרות ← 📱 הגדרות Telegram

```
📍 הגדרת Bot Token
📍 הגדרת Chat ID (זיהוי אוטומטי זמין)
📨 שליחת הודעת בדיקה  (שולחת גם לטלגרם אוטומטית)
🔧 הגדרת הבוט אוטומטית
🧪 בדיקה מקיפה  (חדש!)
```

#### 🔧 הגדרה אוטומטית של הבוט

לחיצה אחת על **"🔧 הגדר בוט אוטומטית"** מגדירה את כל הבוט דרך ה-API:

| מה מוגדר | ערך |
|----------|-----|
| שם הבוט | `Red Alert Monitor` |
| תיאור קצר | "התרעות צבע אדום בזמן אמת" |
| תיאור מלא | "בוט התרעות צבע אדום בזמן אמת — Red Alert Monitor v5" |
| פקודות | `/start` `/status` `/help` `/mute` `/unmute` `/test` |

#### 🧪 בדיקה מקיפה (חדש בגרסה 5.0!)

כפתור **"🧪 בדיקה מקיפה"** בהגדרות מריץ בדיקה שלמה של כל המערכת:

| שלב | פעולה |
|-----|--------|
| **1** | מאפס מצב פנימי — מאלץ פתיחת מסכים מחדש |
| **2** | מקפיץ **מסך מלא + Overlay** עם התרעת בדיקה |
| **3** | שולח הודעה לטלגרם **ישירות** (ללא תלות ב-`telegram_enabled`) |
| **4** | מציג **חלון תוצאות** עם ✅/❌/⚠️ לכל רכיב |

**פורמט הודעת הבדיקה המקיפה בטלגרם:**
```
🧪 בדיקה מקיפה — Red Alert Monitor v5

🚀  ירי רקטות וטילים
🏙  תל אביב | רמת גן | בני ברק | חולון | גבעתיים
🕐  22:09:14

✅  כל המערכות תקינות!
```

**פורמט ההודעה בהתרעה אמיתית:**
```
🚀  ירי רקטות וטילים
🏙  אשקלון  |  נתיבות  |  שדרות
🕐  14:23:05
```

### 📍 זיהוי מיקום אוטומטי

**בכל הפעלה** האפליקציה בודקת היכן אתה נמצא לפי כתובת IP (ב-background, ללא עיכוב בהפעלה):

```
הפעלת האפליקציה
        ↓
  זיהוי מיקום לפי IP (thread נפרד)
        ↓
  ┌─────────────────────────────────────────┐
  │  אין ישובים שמורים?                    │
  │  → דיאלוג: "זוהה מיקומך — ברקת"       │
  │    לחץ ✓ להוספה קבועה                  │
  ├─────────────────────────────────────────┤
  │  יש ישובים שמורים + אתה בישוב אחר?    │
  │  → הודעת Tray: "נמצאת ב-ברקת —        │
  │    נוסף זמנית לרשימה"                  │
  │    הישובים הקבועים שלך פעילים תמיד!   │
  ├─────────────────────────────────────────┤
  │  כבר בישוב קבוע שלך?                   │
  │  → ✅ הכל תקין, אין פעולה נדרשת        │
  └─────────────────────────────────────────┘
        ↓
  כל 5 דקות: בדיקת שינוי מיקום
  (אם עברת מקום → הודעה ועדכון אוטומטי)
```

| **מסך מלא** | פתיחה אוטומטית של FullScreen בהתרעה |
| **מפה** | פתיחה אוטומטית + סגירה אחרי 10 שניות (תמיד) |
| **Google** | שיתוף מיקום עם Google Maps |
| **זמן מסך מלא** | סגירה אוטומטית אחרי N שניות |
| **זמן חלון מרחף** | הצגת overlay למשך N שניות |
| **קצב בדיקה** | כל כמה שניות לשאול את OREF (1–10 שניות) |
| **📱 Telegram** | Token + Chat ID + **🔧 הגדרה אוטומטית** + **🧪 בדיקה מקיפה** |
| **🌐 Webhook** | כתובת URL לקבלת POST JSON בכל התרעה |
| **הפעל עם Windows** | Scheduled Task + Registry autostart |

## 🛠️ פתרון בעיות

### האפליקציה קורסת ללא הודעה
קריסת C++ של Chromium כשרצים כ-Administrator עם GPU מופעל.
**פתרון:** מטופל אוטומטית — הדגלים הבאים מוגדרים לפני `QApplication`:
```
QTWEBENGINE_CHROMIUM_FLAGS = --disable-gpu --disable-software-rasterizer --no-sandbox
QTWEBENGINE_DISABLE_SANDBOX = 1
```

### מקבל התרעות מכל הארץ
ללא ישוב מוגדר — מתקבלות **כל** ההתרעות בישראל.
**פתרון:** פתח **הגדרות** ← לחץ **📍 בחר ישובים** ← הוסף את הישוב שלך.

### התרעה מגיעה באיחור של כמה שניות
**סיבה:** קצב הבדיקה (poll interval) מוגדר גבוה מדי.
במקרה גרוע, ההתרעה עלולה להגיע עד `poll_interval` שניות אחרי שפורסמה ב-OREF.
**פתרון:** פתח **הגדרות** ← **"🔁 בדיקת התרעות כל:"** ← שנה ל-**1 שנייה** למינימום איחור.

### זיהוי מיקום לא עובד
- בדוק חיבור לאינטרנט — זיהוי מיקום דורש חיבור פעיל
- אם הישוב שזוהה שגוי — הוסף את הישוב הנכון ידנית בהגדרות
- הישוב שנוסף זמנית יימחק בפעם הבאה שתיסע למקום אחר

### לא מגיע עדכון "תוצאות נפילה"
- עדכוני החדשות מגיעים **2 דקות** לאחר ההתרעה (גם בטלגרם)
- אם החלון לא מופיע — בדוק חיבור לאינטרנט
- לא תמיד ניתן לאתר ידיעות רלוונטיות — הישוב יכול לא להופיע בכותרות
- גם כאשר לא נמצאו ידיעות, תגיע הודעת "לא נמצאו עדכונים" לטלגרם כאישור שהמערכת פעלה

### Telegram לא עובד

**סיבות נפוצות (לפי סדר חשיבות):**

**1. הטלגרם לא מופעל בהגדרות**
ברירת המחדל היא כבוי. חייב לסמן ✅ את **"הפעל Telegram"** ולשמור.

**2. סינון ישובים חוסם את שליחת ההודעה**
אם הגדרת ישובים ספציפיים — ההתרעה נשלחת לטלגרם **רק** אם הישוב שלך ברשימה.
התרעה על ישוב שאינו ברשימה לא תישלח לטלגרם כלל.
**פתרון:** ודא שהישוב שקיבל התרעה נמצא ברשימת הישובים שלך, או רוקן את הרשימה לקבלת כל ההתרעות.

**3. Token / Chat ID שגויים**
שגיאות נבלעות בשקט — האפליקציה לא מציגה שגיאה אם ה-Token לא תקין.

**שלבי בדיקה:**
1. פתח **הגדרות** ← **📱 הגדרות Telegram**
2. ודא שה-checkbox **"הפעל Telegram"** מסומן ← שמור
3. ודא שה-Bot Token נכון (מתחיל ב-`123456789:ABC...`)
4. לחץ **🔍 זהה אוטומטית** לאחר שליחת הודעה לבוט (לקבלת Chat ID)
5. לחץ **📨 שלח הודעת בדיקה** לבדיקת חיבור
6. לחץ **🔧 הגדר בוט אוטומטית** להגדרת שם, תיאור ופקודות הבוט
7. לחץ **🧪 בדיקה מקיפה** לדוח מלא — יציג בדיוק מה עובד ומה לא

### מסך "מותר לצאת" מוצג באדום
תוקן בגרסה 5.0 — עדכן לגרסה האחרונה.

### המפה לא נסגרת אוטומטית
תוקן בגרסה 5.0 — המפה נסגרת תמיד לאחר 10 שניות, ללא תלות בהגדרות.

### האפליקציה לא עולה עם Windows
**בדיקה מהירה:**
1. פתח Task Scheduler (`taskschd.msc`)
2. חפש "RedAlertMonitor" תחת Task Scheduler Library
3. בדוק שה-Actions מצביע ל: `C:\RedAlertIDF\RedAlertMonitor.exe`

**אם הנתיב שגוי:** הרץ `install.bat` כמנהל.

### לא מגיע סאונד
- בדוק שהסאונד לא מוגדר כ-`שקט` בהגדרות
- בדוק שהאפליקציה אינה במצב **השתקה** (כפתור 🔇 אדמוני בווידג'ט)
- בדוק שהרמקולים/אוזניות מחוברים ומופעלים

### שיתוף מיקום Google לא עובד
- לחץ **🌍 התחבר לגוגל** בהגדרות ← התחבר בדפדפן המוטמע
- אם פג תוקף: לחץ **🔓 התנתק** ← התחבר מחדש
