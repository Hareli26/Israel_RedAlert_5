# RedAlert IDF — ערכת התקנה ל-Windows

> ⚠️ **חשוב:** קובץ EXE של Windows חייב להיבנות על מחשב Windows
> (אי אפשר לקמפל אותו מ-Mac). התיקייה הזו מכילה את כל מה שצריך —
> מעתיקים אותה למחשב Windows ומריצים קובץ אחד.

## מה בתיקייה

| קובץ | תפקיד |
|------|--------|
| `build_exe.bat` | **זה הקובץ שמריצים** — בונה ומתקין הכל אוטומטית |
| `red_alert.py` | קוד המקור של האפליקציה (גרסה מעודכנת עם תיקון ה-API) |
| `requirements.txt` | רשימת ספריות Python |
| `RedAlertIDF.ico` | אייקון האפליקציה |
| `launch.vbs` | מפעיל שקט (ללא חלון קונסול) |
| `הפעל.bat` | הפעלה ישירה מהקוד (ללא בנייה) |

## התקנה — 3 שלבים

1. **התקן Python** (אם אין): הורד מ-https://www.python.org/downloads/
   — בזמן ההתקנה סמן ✅ **"Add Python to PATH"**.
2. **לחץ פעמיים על `build_exe.bat`** — הסקריפט יתקין את התלויות, יבנה את
   ה-EXE ויתקין אותו ל-`C:\RedAlertIDF` (לוקח כ-3-5 דקות).
3. **הפעל:** `C:\RedAlertIDF\RedAlertMonitor.exe` — בהגדרות (גלגל שיניים)
   אפשר לסמן "הפעל עם Windows".

## הפעלה מהירה בלי לבנות EXE

אם רק רוצים להריץ (בלי קובץ התקנה): לחץ פעמיים על `הפעל.bat` —
הוא מריץ את האפליקציה ישירות מהקוד.
