## קבצים

| קובץ | מה זה | איך מתקינים |
|------|--------|--------------|
| `RedAlertIDF-5.3.0.pkg` | מתקין רשמי | לחיצה כפולה → המשך → התקן. האפליקציה תותקן ל-Applications |

(תיקיית `_dist` היא תוצר הבנייה הגולמי — אפשר למחוק אחרי ההתקנה.)

## אם macOS חוסם את הפתיחה

האפליקציה חתומה בחתימה עצמית (ad-hoc), לכן בפתיחה ראשונה:

- **הגדרות מערכת → פרטיות ואבטחה → גלול למטה → "פתח בכל זאת"**
- או: Control + לחיצה על האפליקציה → **פתח**
- או בטרמינל: `xattr -cr /Applications/RedAlertIDF.app`
